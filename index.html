End-to-End Workflow

Step 1

Screaming Frog Crawl + Content Export

  • Crawl selected Financial Express article URLs.
  • Filter article pages using FE-specific include/exclude regex rules.
  • Export URLs and metadata for Python-based content extraction.
  • Keep the crawl clean so weak/non-article pages do not enter validation.
Step 2

Google Cloud NLP Entity + Sentiment Extraction

Google NLP extracts entities, topical signals, salience, article-level sentiment score, sentiment name, and magnitude for each Financial Express article.

Step 3

Claude Sonnet 4.6 Editorial Validation

Claude validates candidate links using source/target context, Google NLP entities, editorial controls, and the strict rule that keyword-only or entity-only matches must be rejected.

Step 4

Vercel Editorial Dashboard

Final validated recommendations are deployed into this dashboard so SEO and editorial teams can review links, filter recommendations, check reasons, and implement internal links.
